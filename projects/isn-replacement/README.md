# ISN Replacement

Replacing Inspection Support Network (ISN) for Safe House Property Inspections.

## Why

- >$12K/year and rising.
- Vendor unresponsive to improvement requests.
- Parent corp shows interest in Safe House client data. Data sovereignty risk.

## Build platform

Replit.

## Slice order

1. **Scheduling and calendar** (active)
2. Agreement and payment intake
3. Inspector dispatch
4. Report delivery
5. Realtor portal
6. Financial reporting
7. Automations

## Work tree

- `discovery/` raw exploration of ISN, screenshots, exported data, field maps.
- `spec/` extracted requirements, workflows, data model per slice.
- `decisions/` ADR-style notes on tradeoffs.
